# Structure du projet

- assets/ : Fichiers CSS, JS, et images.
- auth/ : Fichiers connexion/inscription et déconnexion
- classes/ : Livres et Bibliothèque
	--> Bibliotheque.php : méthodes pour la gestion globale de la bibliothèque
	--> Livres.php : méthodes pour obtenir et définir les propriété des livres
- config/ : connexion à la Base de données 
- documentation/ : Diagramme, SQL et maquettes.
- templates/ : header et footer
- user/ : gestion profil utilisateur
- views/ : 
	--> list.php : liste des livres
	--> addBook.php : ajouter un livre
	--> update.php : modifier un livre
	--> delete.php : supprimer  un livre
	--> search.php : rechercher un livre
	--> details.php : détails d'un livre
- index.php : Page d'accueil du projet.
